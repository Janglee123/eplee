## Installing

Grab the latest release [here](https://github.com/Janglee123/eplee/releases)

## Development

Clone this repository

```bash
git clone https://github.com/janglee123/eplee
```

change directory to the cloned path

```bash
cd eplee
```

Install dependencies

```bash
yarn
```

Start app

```bash
yarn run dev
```
